# THREAD_ARCHIVE

## Public Thread Policy

This file is safe for a public repository. Real thread IDs, local paths, session titles, and detailed recovery notes belong only in local private records.

## Archive Rules

- Old V05/R6 threads are archived and cannot be used as V06 dispatch targets.
- V06 starts with no active A-F production threads.
- A new role thread may be created only after the user explicitly starts a V06 small-sample test.
- Each role may have at most one active thread.
- If a thread fails, Z records a public-safe summary here and keeps concrete IDs in local private records.

## Public Status

| Scope | Status | Note |
|---|---|---|
| V05/R6 historical threads | archived | Not a dispatch target |
| V06 Z orchestration | framework ready | Manual start only |
